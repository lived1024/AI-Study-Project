## Grounding
사실에 기반한 답변

### Grounding 주요방식 - RAG
- RAG(Retrieval Augmented Generation)
  - Retrieval   : 쪽지를 만들어서
  - Augmented   : (생성하기 전에 LLM에게 패스하여) 더 나아진
  - Generation  : 생성

### RAG 방법
- RAG 버전 1
  - 프롬프트에 데이터 포함
  - API call로 데이터 가져오기
- RAG 버전 2
  - 미리 저장되어 있는 데이터를 가져와서 프롬프트를 만듬 - embedding
  - LLM 모델을 몇 번 쓸 수도 있음
- RAG 버전 3
  - Vector DB를 만들고 데이터를 저장
  - Embedding으로 데이터를 저장하고 찾을 때도 씀
- RAG 버전 4
  - Fine Tuning을 통한 LLM 서비스 자체를 업그레이드
