- 회사에서 허용하지 않는 경우
- 잘 알려지지 않은 언어
- 복잡성을 더하는 문제 : 그럴듯하나 교묘하게 틀림
- "생산적"이라는 착각
- 테스트 데이터의 품질 문제
  - LLM 자체 생성한 경우도 많음
  - 그럴듯한 경우도 있음
- 코드를 어디에서 가져오는가의 문제
  - 저작권
  - 틀린 코드 등
